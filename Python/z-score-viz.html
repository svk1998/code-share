How It Works

Understanding the anomaly detection logic

📊

What is a Z-Score?

Z-score tells you how far this week's count is from the historical average, measured in standard deviations. The further it is, the more unusual it is.

Z = (this_week − avg) / std_dev
🎯

Alert Thresholds

✅ Z ≤ 2 — Normal. Expected range.

⚠️ 2 < Z ≤ 3 — Warning. Elevated. Worth reviewing.

🚨 Z > 3 — Critical. Rare spike. Act now.

🔄

Baseline Updates

The average and std dev are recomputed every week using the last 12 normal weeks. Anomalous weeks are excluded — so spikes don't corrupt your baseline.

window = last 12 non-anomaly weeks
Z-Score Visual Scale
−4 (unusually low) −2 0 (normal) +2 +4 (spike 🚨)
Normal — safe to update baseline
Warning — manual review
Critical — excluded from baseline
